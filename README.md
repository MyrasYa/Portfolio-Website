## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension (optional, for VS Code users)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/MyrasYa/portfolio-web.git
   ```

2. **Navigate to project directory**

   ```bash
   cd portfolio-web
   ```

3. **Open in browser**

   **Option A: Direct opening**
   ```bash
   # Windows
   start index.html
   
   # macOS
   open index.html
   
   # Linux
   xdg-open index.html
   ```

   **Option B: Using Live Server (Recommended)**
   - Open the project folder in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"

### Navigation

- Use the hamburger menu (â˜°) to access different sections

### Sections

3. **Styling**: Modify colors and fonts in `style.css`
4. **Skills**: Add your own skills in the Skills section
5. **Social Links**: Update GitHub and LinkedIn URLs
